## Description

基于 Nestjs 开发的，前端 Node 服务

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


这是一个微信小程序项目的后端代码，收纳小助手。

## Installation

```bash
$ npm install
```

## Running the app

```bash
# 本地开发
$ npm run start

# 本地开发监听模式
$ npm run start:dev

# 线上部署
$ npm run build && npm run start:prod
```
